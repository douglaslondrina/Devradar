# DevRadar 

DevRadar is a simple Full-stack demo App, created as part of a short React Bootcamp offered by the cool guys at RocketSeat.

## Running

You will need at min. Node 12 and yarn.

Run yarn on all three folders: backend, web and mobile. And then, `yarn start` on each. The App is connected to my free MongoDB Atlas instance and it may stop to work. To change the URL inside backend/src/index.js file.
